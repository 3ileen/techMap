### Backend & Database
- Extracted data is stored in **Supabase (Free Tier)**.

### Frontend
- Built with **Vite + React + TypeScript**
- Styled using **Tailwind CSS** and **shadcn/ui**
- Initial UI scaffolding created with **Lovable**, then manually extended and connected to Supabase.

### Learning Resources
- Simple redirection system linking skills to **Udemy searches**.
- Designed to support multiple platforms in future versions.

---

## Tech Stack

- **Frontend:** Vite, React, TypeScript, Tailwind CSS, shadcn/ui  
- **Backend / Database:** Supabase  
- **Data Processing:** Python  
- **Version Control:** GitHub

## Why SKILLBRIDGE?

SKILLBRIDGE helps users focus on **what really matters**, saving time, reducing confusion, and making tech career paths clearer and more accessible—especially for underrepresented and Spanish-speaking communities.
